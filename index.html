Local Swap Solo

Github


A Ruby on Rails applications designed to connect users who want to trade goods with other users in the same geographic area by allowing users to create profiles in a geographically and socially enabled environment. Cragslist has become a ubiqutious platform and functions effectively although it suffers from a number of scams and quite frankly creepy people. Local Swap is still a work in progress, but ultimately will have users authenticating through facebook in order to create a safer community. User locations are hidden via a location scrambling function that offsets their given location by up to 500 ft. Directions are available for users to locate other users, but those directions will never point to a user's true location. In this way user locations are used as an asset to the application but not made available for other users.

Features: Ruby on Rails, PostgreSQL, Authentication using the Devise Gem, Photo Uploads and Image Cropping, Mailboxer Gem messaging and emailing, React-Rails Gem, Geocoder Gem, Google Maps Javascript API, Google Maps Directions API, Seeded data via Faker Gem.

A Ruby on Rails applications that makes heavy use of the React-Rails Gem to provide a streamlined experience for users that want a guarded image management platform. Portland Weird Map Group

Github


The Portland Weird Map is a ember.js applocation that creates geographical display of "weird" locations around Oregon's biggest city. The Portland Weird Map allows users to quickly become familiar with the locations and descriptions of Portlands most iconic locations. In addition to typical features present in Google Maps API applications (zooming, scrolling, street, and satellite views), the Portland Weird Map allows users to interact with how data points are displayed with category filtering and the ability to add more weird locations. All data is stored and retrieved using Google's Firebase. The ability to route a bike map between 2 and 5 locations is also available to the user. Lastly, the map contains a Data Map that shows a color coded cluster map of bars in Multnomah County by neighborhood.

Features: Ember.js, Google Firebase, Google Maps Javascript API, Google Maps Directions API

Apocalypse App Group

Github


The Apocalypse App is a sinatra with postgresql built application that demonstrates many types of activerecord relationships between its object classes in order to create a turn-based, textually centric game. Users will stock up on resources and see how long they will survive in a given apocalyptic scenario with their given supplies. Random events will be triggered as users progress through the game and at times choices will become available that will serve to either help or hinder the user's progression. User high scores are persisted in the database.

Features: Sinatra, PostgreSQL, Hazmat Suits
